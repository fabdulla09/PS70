<br> #### Week 6 & 7: Input and Output Week <p1> I wanted to make something that could potentially help with my final project, so I ended up making an input and output device using a phototransistor to where if it reads below a certain reading then the light would turn off. The code segment is placed below <p1> <pre> <code> const int photo = A0; // variable which stores pin number const int pin = 9; void setup() { Serial.begin(9600); pinMode(photo, INPUT); //configures pin 3 as OUTPUT pinMode(pin, OUTPUT); } void loop() { int value = analogRead(A0); Serial.println(value); if (value < 300)// the point at which the state of LEDs change { digitalWrite(pin, LOW); //sets LEDs ON } else { digitalWrite(pin,HIGH); //Sets LEDs OFF } } </code> </pre>
